# 链接与 Logo 迁移工具

一个用于 WordPress 网站之间迁移“链接/书签”的独立插件。支持按分类筛选、逐条选择、搜索、字段开关、Logo 本地化、重复覆盖、进度提示和详细失败报告。

作者：[Mao TK](https://www.maotk.com/)

## 功能

- 按链接分类筛选，包括“未分类”链接。
- 搜索链接名称、网址和简介。
- 逐条勾选链接，支持全选、取消全选、选择当前结果。
- 可选择是否迁移 Logo、简介、评分和分类。
- 始终迁移名称、网址、可见性、打开方式、关系、备注和 RSS 地址。
- 将 Logo 打进 ZIP，并在新站导入媒体库。
- 支持 PNG、JPEG、GIF、WebP、AVIF、BMP、ICO 和安全清理后的 SVG。
- 根据图片内容识别格式，不依赖错误或缺失的扩展名。
- 按图片内容哈希去重，重复导入不会反复创建相同媒体。
- 可覆盖网址相同的已有链接。
- 显示导出/导入进度、耗时和导出完成提示。
- 分别报告失败链接、失败 Logo 和其他警告。

## 下载与安装

最新可安装包：

[`dist/wp-link-logo-migrator-2.1.2.zip`](dist/wp-link-logo-migrator-2.1.2.zip)

安装步骤：

1. 登录 WordPress 后台。
2. 打开“插件 → 安装插件 → 上传插件”。
3. 选择 ZIP 文件并点击“现在安装”。
4. 安装完成后启用插件。
5. 从“工具 → 链接与 Logo 迁移”进入插件。

也可以把仓库中的 `wp-link-logo-migrator` 文件夹上传到：

```text
wp-content/plugins/
```

## 完整迁移教程

### 一、在旧网站导出

1. 在旧站安装并启用插件。
2. 打开“工具 → 链接与 Logo 迁移”。
3. 在“选择迁移内容”中决定是否包含：
   - Logo 图片
   - 简介
   - 评分
   - 链接分类
4. 使用分类筛选缩小列表范围。
5. 使用搜索框查找名称、网址或简介。
6. 勾选最终需要迁移的链接。
7. 点击“下载链接迁移包”。
8. 等待进度窗口显示“导出完成”，浏览器会下载 ZIP。

分类和搜索只负责控制当前可见列表，最终以每行复选框是否勾选为准。

### 二、备份新网站

导入前建议备份：

- WordPress 数据库
- `wp-content/uploads`
- 当前链接数据

可使用 WPvivid、主机快照或数据库管理工具。

### 三、在新网站导入

1. 在新站安装并启用同一插件。
2. 打开“工具 → 链接与 Logo 迁移”。
3. 上传旧站生成的迁移 ZIP。
4. 建议勾选“覆盖网址相同的已有链接”。
5. 点击“开始导入”。
6. 等待页面自动返回导入报告。

导入报告会显示：

- 新增链接数量
- 覆盖链接数量
- 跳过链接数量
- 新增 Logo 数量
- 失败链接及原因
- 失败 Logo、原地址及原因
- 分类等其他警告

## 覆盖规则

插件优先按完整网址精确匹配已有链接；找不到时，再按规范化后的网址匹配。

勾选覆盖后：

- 更新名称、网址、可见性和打开方式。
- 仅更新导出时选中的简介、评分、分类和 Logo。
- 未选中的可选字段尽量保留新站现有数据。
- 新 Logo 成功导入后替换旧 Logo。
- 新 Logo 失败时，已有链接会优先保留原 Logo。

不勾选覆盖时，网址相同的链接会跳过。

## Logo 处理

- 可访问的 Logo 会放进迁移 ZIP。
- 导入后 Logo 位于新站媒体库，不再依赖旧站。
- 内容相同的图片会复用已有媒体。
- SVG 会移除脚本、事件属性、外部资源和危险元素。
- 无法下载的 Logo 会在报告中显示旧地址和失败原因。

## 服务器要求

- WordPress 5.8 或更高版本
- PHP 7.4 或更高版本
- PHP `ZipArchive` 扩展
- SVG 处理需要 PHP DOM 扩展
- 服务器能够访问旧站或外部 Logo 地址

大批量迁移时，建议提高：

```ini
memory_limit = 256M
max_execution_time = 300
upload_max_filesize = 512M
post_max_size = 512M
```

实际配置方法取决于主机环境。

## 常见问题

### 为什么 Logo 没有打包？

常见原因包括防盗链、图片地址失效、远程服务器拒绝访问、文件超过限制或格式无法识别。导入报告会保留具体原因。

### 可以重复导入吗？

可以。勾选覆盖后，重复导入可用于更新链接或修复之前失败的 Logo。

### 会覆盖文章、用户或网站设置吗？

不会。本插件只处理 WordPress 链接表、链接分类关系和相关媒体附件。

### 百分比和预计时间如何计算？

百分比按服务器实际完成的链接任务计算。预计时间根据已经完成任务的平均速度动态更新；单个大 Logo 下载较慢时，预计时间可能暂时波动。

达到 100% 后会显示右上角和底部关闭按钮。窗口不会自动消失，必须由用户手动关闭。

## 安全建议

- 只导入由本插件生成且来源可信的 ZIP。
- 导入前备份新站。
- 不要手动修改迁移包中的 `manifest.json` 或图片路径。
- 迁移结束后抽查链接、分类和 Logo。
- 如果迁移包包含敏感备注，请勿公开分享该 ZIP。
