在两个 WordPress 网站之间迁移链接、链接分类、简介、评分和 Logo 图片。

== 使用方法 ==

1. 在旧站安装并启用插件。
2. 打开“工具 → 链接与 Logo 迁移”，点击“下载迁移包”。
3. 在新站安装并启用同一个插件。
4. 打开“工具 → 链接与 Logo 迁移”，上传刚才下载的 ZIP。
5. 根据需要选择是否更新网址相同的已有链接，然后开始导入。

== 说明 ==

* Logo 图片会放入迁移 ZIP，并在新站上传到媒体库。
* 支持安全清理后的 SVG，以及无扩展名或扩展名错误的常见图片。
* 再次导入时可覆盖网址相同的链接，并复用内容相同的 Logo，避免重复媒体文件。
* 无法下载的远程 Logo 会保留原始图片 URL。
* 导入前建议使用 WPvivid 或主机面板备份新站数据库。
* 服务器必须启用 PHP ZipArchive 扩展。
